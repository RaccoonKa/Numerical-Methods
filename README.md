# Numerical-Methods
<div align="center">  
Численных методов властитель      
И повелитель всех сеток —      
Явился к нам наш вдохновитель:      
«Рисуй, студент, график исток!»  

То 2D взовьется круто,    
То 3D впадет в зенит —    
А лектор, сбросив с плеч усталость,    
В театр свой спектакль манит.  

«Живите ярко! — даст совет. -      
Как в 3D, вершите судьбы, дети!»    
Один лишь в том сомненья нет:    
Все методы — численны на свете.  
</div>
